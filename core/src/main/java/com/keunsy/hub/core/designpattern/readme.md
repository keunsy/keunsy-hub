## 23种常见设计模式

##  六大设计原则(SOLID)
- 单一职责原则(SRP)
- 开闭原则(OCP)
- 里氏替换原则(LSP)
- 迪米特法则(LOD)
- 接口隔离原则(ISP)
- 依赖倒置原则(DIP)

### 易记口诀
#### 三大分类：
- 建造型： 抽工单减员 （抽象工厂模式、工厂方法模式、单例模式、建造者模式、原型模式）
- 结构型：桥想呆世外组装（接桥模式、享元模式、代理模式、适配器模式、 外观模式、组合模式、装饰模式）
- 行为型：芥末被叠放至名册中观状（解释器模式、模板方法模式、备忘录模式、迭代器模式、 访问者模式、职责链模式、命令模式、策略模式、 中介者模式、观察者模式、状态模式）
